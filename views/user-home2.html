A cloud based analytical tool to predict the likelihood of future occurrences. Track the events and upload the data to find the result of an event. An alarm which wakes you up before its late.

Overview

  • The future is an undiscovered country. It is unknown and unknowable.
  • And yet, we can shape it. We can prepare. We can dream.
  • PREDICT ME is a platform for our clients to predict the future occurrences of an event by feeding the system with historical data.
  • It is amalgamation of sheer hard work and dedication, Machine learning, lots of maths, and continuous effort of out team.

Generation of Keys

  • By clicking on the Generate Key button in the Keys tab.
  • Two keys namely appKey & secretKey will be generated.
  • The keys will be unique for each registered account. So the same keys will be used for all domains & entities of the account.
  • The keys have to be included in the request headers while making a request to any of the REST-API of PredictMe
We've put together this convenient and effective Pictorial Index to the PREDICT ME App for your convenience.

Creating Domain

  • A domain is a generic field of study, to carry on predictive analysis.
  • Example: 'Health Sciences'
  • The domains are uniquely identified by name.
  • Note : Special characters are restricted from using in domain name.

Creating Entity

  • An entity is a sub-field of the domain.
  • Example: 'Heart Stroke'
  • The entities are uniquely identified by name.
  • Note : Special characters are restricted from using in entity name.

Uploading Headers for the domain & entity combination

  • The json to be posted for uploading headers should be of the format :
  • { "headers" : [ { "index" : 0, "attributeName" : "Height", "attributeType" : "numeric" }, { "index" : 1, "attributeName" : "Name", "attributeType" : "nominal", "attributeValues" ["Name1", "Name1", "Name1"] } ] }

Upload Historical data for the domain & entity combination.

  • The json to be posted for uploading historical data should be of the format :
  • { "data" : [ { "datum" : "a1,a2,a3,a4,.....,an,t" } ] }

Upload Prediction data for the domain & entity combination.

  • The json to be posted for uploading prediction data should be of the format :
  • { "data" : [ { "datum" : "a1,a2,a3,a4,.....,an," } ] }
